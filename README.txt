*Sample RESTful FUSE implementation*

End goal of this project is to have an implementation of libFUSE which takes
advantage of REST to get file system information from a remote host.

This project requires:
- [libFUSE][https://github.com/libfuse/libfuse]
- [restclient][https://github.com/mrtazz/restclient-cpp]
- C++23
- CMake
- Makefile
